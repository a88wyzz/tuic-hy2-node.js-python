# 1.Hysteria2在Nodejs/Python一键脚本极简部署（Pterodactyl 翼龙面板）

* 更新自适应端口，无需再手动设置

* Hysteria2版本：2.6.5

```
curl -Ls https://raw.githubusercontent.com/a88wyzz/tuic-hy2-node.js-python/main/hy2.sh | sed 's/\r$//' | bash
```
